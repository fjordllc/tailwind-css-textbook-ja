# 第5部 実践的な Tailwind CSS

第4部までで、画面を組むための道具がそろいました。この第5部では、それらを使って**実際のプロダクトで必ず求められるテーマ**に取り組みます。レスポンシブ（[第17章](chapter17.md)）、ダークモード（[第18章](chapter18.md)）、アニメーション（[第19章](chapter19.md)）、フォーム（[第20章](chapter20.md)）、アクセシビリティ（[第21章](chapter21.md)）です。

これらはすべて、第2部で学んだ**バリアントの仕組み**（[第6章](../part2/chapter6.md)）の応用です。`md:`・`dark:`・`hover:`・`motion-reduce:`・`invalid:` といったバリアントが、ここで実戦投入されます。「条件付きでスタイルを当てる」という Tailwind の核心が、最も生きる場面です。
